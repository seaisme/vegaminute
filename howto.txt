** เซ็ต git **
1.เปิด git bash
2.ตั้งชื่อกับเมลล์ผู้ใช้ ให้ใส่ชื่อกับเมลล์ที่สมัครgithub
git config --global user.name "ชื่อ"
git config --global user.email "เมลล์"
3.ตรวจสอบเมลล์กับชื่อ
git config --global --list


**วิธีอัพโหลดไฟล์ลงนะ**
1.เปิด git bash
2.ไปที่เก็บไฟล์ของตัวเอง cd ตามด้วยที่อยู่ไฟล์ เช่น cd C:/xampp/htdocs/vegaminute
พอเข้าที่เก็บไฟล์ตัวเอง
3.ให้พิมพ์ git add . เพื่อแอดไฟล์งานตัวเอง
4.แล้วพิมพ์ git commit -m "ใส่คำอธิบายกำกับ เช่น index complete" 
เพื่ออธิบายการลงงานของตัวเองว่าอัพอะไรลงไป
5.อัพโหลดขึ้น github ด้วยคำสั่ง git push origin dev
(ข้าวปุ้น git push origin master)
